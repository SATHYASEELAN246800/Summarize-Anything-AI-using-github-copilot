# Summarize Anything AI

A comprehensive multi-modal summarization platform using Hugging Face models.

## Features

- Multi-platform support for various input types:
  - Videos (YouTube, Drive, social media)
  - Audio (MP3, WAV, podcasts)
  - Documents (PDF, DOCX, PPTX)
  - Text and URLs
  - Images (OCR)

- Advanced processing:
  - Transcription
  - Multiple summary types
  - Chapter extraction
  - Quiz generation
  - Sentiment analysis
  - Multi-language support
  - Translation
